### Wizards
In addition to rendering forms, this library can also be used to render complex wizard workflows.

```js
var wizard = new FormioWizard(document.getElementById('wizard'));
wizard.src = 'https://examples.form.io/wizard';
```

```html
<div id="wizard"></div>
```
